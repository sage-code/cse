Learn HTML

HTML stands for Hyper Text Markup Language. It is a data oriented language and looks like XML. Both languages HTML and XML are markup languages based on tags. These are keywords enclosed in angle brackets. The difference is, HTML is design specific for documents while XML was design to transfer text based data.
