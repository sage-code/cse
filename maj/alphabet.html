Maj Alphabet

Maj has an alphabet derived from ISO Latin. It has 26 letters associated with 30 sounds. The language has a balanced consonant-to-vowel ratio of 3:1 (21 consonants to 7 vowels). We avoid consonat clusters and we use simple word patterns to make Maj words easier to pronounce. Recently we have reintroduced W and Q to keep Maj alphabet compatible with English. Because now you can represents all English sounds.

Notes: 

  • Each letter has an uppercase version;
  • We use "W" like a soft "U" that can't get accent;
  • We use "Q" with "w" or "u" after it always and is pronounced /kw/ or /ku/;

Maj Script

Maj script is created by reusing Unicode letters from numerous alphabets including Lating, Greek and Georgian. Each letter is design to be created from a single pen stroak. Maj script is written with a pen or a stylus, using a continuous coursive line.

Maj script letters have round portions, sharp angles and returns. You must learn where to start a letter and go ahead in a single direction until you finish the letter. Between letters there is a short distance. Between words, the distance is larger, at least double, to distinguish the independent words.

Pronunciation guide

Maj pronunciation is based on deterministic rules. That means, pronounciation rules are the same in all words and do not depend on the word origin. In Maj you do no need to learn how to spell words. That is if you know Maj alphabet and reading rules then you can read any Maj word correctly even if you have never heared that word before.

Notes

  • Maj alphabet has 7 vowels, 19 consonants and 4 digraphs in total 30 sounds;
  • Maj sounds require healthy tongue, lips, teeth and vocal chords;
  • Maj sounds do not require to put your tongue between your teeth;
  • Maj sounds do not require inverted tongue tip (retroflex);
  • Maj has strong consonants and clear vowels similar to Portuguese;
  • Maj sounds have a Balkanic influence, though not all sounds are derived from Latin.

Modified Letters:

  1. "h" is read /ə/ like "eo" in word "pigeon";
  2. "j" is read /ʒ/ like "su" in word "pleasure";
  3. "x" is read /h/ like "ho" in word "hollywood";
  4. "c" is read /ʃ/ like "sh" in word "sheep";

Pronounciation

Vowels are pronounced like in Italian and Romanian languages not like in English nor German:

  • "a" read /a/ is open central, like "a" in "car";
  • "e" read /e/ is close-mid front, like "e" in "melon";
  • "i" read /i/ is close front, like "ee" in "geese";
  • "o" read /o/ is close-mid back, like "o" in "document";
  • "u" read /u/ is close back, like "oo" in "soon";
  • "y" read /y/ is very similar to sound "i" in words "still"
  • "h" read /ə/ is mid central, like "o" in baron;

Tone and accent

Maj is designed for chat, so it has no official tone. When can be read with a monotone voice by a robot and this is fine. It is not wrong to use tone to improve the meaning of long statements, enumerations, queries or exclamative sentences but this is unusual and characteristic to humans that have a native tonal language. Maj do not require use of any tonal or accent for speaking the words. Accent on first or second vowel will just do ok. However we do not put the accent on the last vowel.

Pronounciation

Most consonants are similar to IPA but some are different;

  • "b" is voiced, plosive, bilabial like "b" in "boss";
  • "c" is voiceless, post-alveolar fricative like "sh" in "fashion";
  • "p" is voiceless, plosive, bilabial like "p" in "pin";
  • "t" is voiceless, plosive, alveolar like "t" in "test";
  • "d" is voiced, plosive, alveolar like "d" in "door";
  • "k" is voiceless velar stop like "k" in "kitten" or "c" in "color";
  • "m" is nasal, bilabial, like "m" in "mule";
  • "n" is alveolar, nasal, like "n" in "novel";
  • "r" is trilled, similar to sound "r" in Spanish "perro";
  • "f" is voiceless, labio-dental fricative like "f" in "frog";
  • "v" is voiced labio-dental fricative, similar to "v" in "voice";
  • "s" is fricative alveolar like "s" in "snake";
  • "z" is alveolar-dental, vibrated, similar to "z" in "zoo";
  • "l" is liquid, alveolar, like "l" in "lord";
  • "g" is voiced velar fricative like "gu" in "guitar";
  • "j" is voiced fricative alveolar, like "s" in "pleasure";
  • "x" is voiced fricative glottal like "h" in "happy";

Digraphs

Digraphs are groups of 2 letters that are thigtly connected and become a new and distinctive sound. In fact we have created special script letters for these 3 sounds. In other languages one vowel and one consonant can make a digraph but in Maj digraphs are created using two consonants.

DIGRAPHLC IPA
Tc tc /t͡ʃ]
Ts ts /t͡s]
Dj dj /d͡ʒ]
Gn gn /ŋ]

Digraph Pronunciation

  • Digraph "tc" sounds "t͡ʃ" like "ch" in "peach"
  • Digraph "ts" sounds "t͡s" like "ts" in "cats"
  • Group "dj" sounds "d͡ʒ" like "g" in "giraffe"
  • Cluster "gn" is read /ŋ/ similar to "gn" in Italian word "signora";

Listen and learn digraphs pronounciation, in order.

{ tc, ts, dj, gn }

Consonant Clusters

A cluster represents a group of two consonants. Maj is using clusters in many words using the pattern: CCV or VCCV.

Rules:

  • We do not use the same letter twice to make clusters;
  • Clusters of 3 consonants are extremely rare in Maj;
  • Some combinations cannot be pronounced, therefore are discouraged;

Examples:

  • group "tz", is actually "ts" (more realistic)
  • group "sd", is actually "st", making "d" after s is too hard
  • group "dz", is not used but group "zd" can be easy used
  • group "pz", is hard to use so we preffer "ps" instead
  • ...there is no point to identify them all now
