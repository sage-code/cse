About Maj School

Maj: is an artificial auxiliary language designed for International fast chat. Maj is inspired from traditional Romance languages, with Latin printed alphabet and Georgian inspired script alphabet. Maj Language

Next we will describe Maj language in details. Maj should be more easy to learn compared to other languages but it will require many hours of study because it has a relative large and rich vocabulary (5000 words). This language is not yet spoken by anybody except the author. Next document was created to introduce Maj language to young learners. Is a free document that you can read on-line:

Maj design was started in 2020. Since then we have created a dictionary and basic grammar book. Our dictionary has about 5000 words and expressions. We will ocasionaly create new words and we create examples to learn the language from short text articles and stories. The language is stabile and comprehensive. We will however probably change some words over time to improve the language flow.
